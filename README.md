# Backend for Topu service

## 環境構築

#### Java 21 設定 
```shell
mise i
```

#### アプリケーションスタート
```shell
mise run up
```

#### Google OAuth2 設定
- `client_id`, `client_secret` は confluenceを 参照

## Architecture
- 最初はレイヤードアーキテクチャー
- 後でヘキサゴナルアーキテクチャーに変更

## Tech
- Java 21
- Spring, Spring Boot
- JPA(Hibernate)
- QueryDSL
- REST Docs With asciidoc
- MySQL
